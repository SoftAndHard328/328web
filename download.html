📋 系統需求

作業系統 Windows 10/11 64位元
記憶體 至少 4 GB RAM
儲存空間 至少 500 MB 可用空間
網路 需要網路連線

🚀 快速安裝步驟

下載程式

點擊上方「下載環境配置工具」按鈕，將 setup.bat 儲存到您的電腦

安裝 DroidCam 手機 App

在您的手機安裝 DroidCam 應用程式（Android/iOS），並確保手機與電腦在同一 WiFi 網路

執行環境配置

執行 setup.bat，它會自動檢查並安裝 Python 3.13 和所有必要套件

取得監控程式

環境配置完成後，從 GitHub 下載最新的 DroidCamAIMonitor.exe 執行檔

💡 提示

setup.bat 會自動檢查系統是否已安裝 Python 3.13，如果沒有會提示您安裝。所有必要套件（OpenCV、YOLO、PyQt5 等）都會自動安裝。
